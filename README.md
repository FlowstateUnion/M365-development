# M365 development

Separate workspace for Microsoft 365 automation and integration work.

Initial focus:

- Copilot Studio
- Power Automate
- Microsoft Teams
- SharePoint
- Microsoft Graph
- PowerShell
- Azure automation and adjacent tooling

Workspace layout:

- `docs/` reference notes, inventories, setup decisions
- `projects/` project-specific workstreams
- `runbooks/` operational procedures and automation notes
- `scripts/` helper scripts for local tooling
- `skills/` repo-local junctions to the installed M365 skill folders for inspection
